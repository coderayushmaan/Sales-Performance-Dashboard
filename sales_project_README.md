# Sales Performance Dashboard 2023
**Project by: Ayushmaan Pandey | BCA, CSJMU Kanpur**
**Role Target: Data Analyst | Business Analyst | MIS Analyst**

---

## Project Overview
An end-to-end Sales Performance Analysis project built on a 1,000-order dataset 
covering 4 regions, 5 product categories, 25 products, and 10 salespeople across 2023.

## Business Problem
The sales team lacked a centralized reporting system to track revenue trends, 
salesperson performance, regional gaps, and category profitability — leading to 
delayed and uninformed business decisions.

## Solution
Built a complete analytics solution including:
- Data generation and cleaning (Python + Pandas)
- SQL-based business queries for KPI extraction
- Multi-chart Python dashboard (Matplotlib)
- Professional Excel workbook with 4 analytical sheets

## Key Findings
- Total Revenue: ₹7.47 Crore | Total Profit: ₹2.63 Crore
- South region leads in revenue contribution
- Electronics is the top-performing category
- Q4 recorded peak quarterly revenue
- 20% discounts reduce profit margins — recommend selective discounting

## Tools Used
- Python (Pandas, NumPy, Matplotlib)
- SQL (MySQL-compatible queries)
- Microsoft Excel (Pivot Tables, Charts, Conditional Formatting)
- Power BI (dashboard recreation recommended)
